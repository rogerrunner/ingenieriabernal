## 🚀 PUBLICAR EN VERCEL — 1 SOLO PASO (SIN GITHUB, SIN COMANDOS)

### Método: Subir directamente en Vercel (5 minutos)

1. **Ir a vercel.com** → crear cuenta gratis con tu email
2. En el dashboard, clic en **"Add New..." → "Project"**
3. Clic en **"Import Git Repository"** — ABAJO busca la opción **"Upload"** o arrastra la carpeta
4. Descomprimir el ZIP → arrastrar la **carpeta `bic`** completa al área de Vercel
5. Vercel detecta Vite automáticamente → clic **"Deploy"**
6. En 2 minutos el sitio está en vivo con una URL temporal

### Conectar el dominio ingenieriabernal.co (5 minutos más)

1. En tu proyecto Vercel → **Settings → Domains → Add** → escribir `ingenieriabernal.co`
2. Vercel te da 2 registros DNS
3. En Namecheap → **Domain List → Manage → Advanced DNS** → agregar esos registros
4. En 10–30 minutos: `ingenieriabernal.co` ya carga tu sitio

### ¿No aparece opción Upload en Vercel?

Usa este método alternativo con GitHub — igual de simple:
1. github.com → crear cuenta → nuevo repositorio privado `bic-web`
2. Subir los archivos del ZIP (arrastrar al navegador)
3. vercel.com → "Add New Project" → seleccionar el repositorio → Deploy
